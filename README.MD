# Prisma notes

## Commands
```bash
# inicjalizacja konfiguracji projektu prisma
npx prisma init --datasource-provider postgresql

# stworzenie modeli na podstawie istniejÄ…cej struktury db
npx prisma db pull   

# reset do pustej bazy
npx prisma db push --force-reset

# wprowadzanie zmian do schemy w db
npx prisma migrate dev --name <init>

# GUI for dc
npx prisma studio
```


## CRUD

```typescript

// Create one

await prisma.<model_name>.create({
    data: {
        <column_name>: "<value>",
    }
});

// Read many

await prisma.<model_name>.findMany();

// read with filter
await prisma.<model_name>.findMany({
    where: {
        column_name: value
        column_name: {
            operator: value
        }
    }
});

// read with additional tables
await prisma.<model_name>.findMany({
    where: {
        column_name: value
    },
    include: {
        relation_name: boolean
    }
});


// read with specific columns
await prisma.<model_name>.findMany({
    where: {
        column_name: value
    },
    select: {
        column_name: boolean
    }
});

```
